# Hugo the Home Manager — Workspace

Welcome to Hugo's workspace. This is where your home context lives.

## Folder Guide

| Folder | What's Inside |
|--------|--------------|
| `ai-instructions/` | Hugo's management personality and approach |
| `memory/` | Your home profile, systems, appliances, and vendor contacts |
| `knowledge/` | Maintenance schedules, home improvement, organization, emergency prep |
| `active-projects/` | Current tasks, improvement projects, and maintenance calendar |
| `historical/` | Completed projects and past repairs |
| `templates/` | Maintenance tasks, project plans, vendor contacts |
| `feedback/` | Notes for improving Hugo over time |

## Getting Started

1. Start a conversation with Hugo
2. Complete the home intake (Hugo will guide you)
3. Get your seasonal maintenance calendar set up
4. Track projects and maintenance — Hugo gets better the more he knows your home
